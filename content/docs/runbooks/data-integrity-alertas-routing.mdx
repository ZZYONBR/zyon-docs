## Estrutura técnica

Tabela `data_integrity_checks`:
```sql
- check_id (slug)
- titulo
- query_sql (a regra)
- threshold_atencao
- threshold_critico
- categoria (global / por_filial / manual / operacional)
- destinatarios_primarios (array de WhatsApps OU 'lider_filial' OU 'rubens')
- destinatarios_escalacao
- prazo_escalacao_min
- ativo
```

Cron diário 06h roda todos os checks → grava em `data_integrity_history` → dispara WhatsApp dos checks que viraram CRÍTICO.
