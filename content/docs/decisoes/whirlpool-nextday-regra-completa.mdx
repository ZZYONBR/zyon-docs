Variações abaixo de R$ 50 indicam **erro de emissão NotFis** (Whirlpool envia
arquivo errado). Esse erro é **cosmético**: muda a proporção CT-e/Carta mas
o líquido recebido fica igual (o complemento entra na Carta de Crédito).

## Reentregas — regra de cobrança

```
Tentativas = 1 → Entrega NextDay normal. Sem cobrança extra (já no CT-e).
Tentativas = 2 → 1 reentrega Ícara. Cobra 1× frete reentrega 60%.
Tentativas = N → (N-1) reentregas. Cobra (N-1)× frete reentrega 60%.

Devolução: mesma regra (cliente recusou após N tentativas).
NF em manifesto NextDay W-HOUSE na 1ª: SEM cobrança extra.
NF em aberto: NÃO entra no mês (só finalizadas/devolvidas).

Tent=0 + ocorrência "Devolução com cobrança" → considera Tent=2 (mín 1 reentr).
```

## Cálculo do valor de cada reentrega

```
m³ = peso_cubado / 167                    (regra Whirlpool)
faixa = lookup na Tabela Super Rápida pelo m³
frete_bruto = MAX(frete_min, m³ × frete_var)
frete_60% = frete_bruto × 0,60
frete_final = frete_60% × (1 + reajuste_diesel_do_mes)
valor_total_nf = (Tentativas-1) × frete_final
```
