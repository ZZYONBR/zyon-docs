## Decisões técnicas tomadas

1. **Driver dual desde já** (`SOFIA_INDEX_DRIVER` env): `local_json` (Sprint 1A) e `pgvector` (Sprint 1B). Switch é trivial.
2. **Contexto de filial+autor+área prepended ao texto do chunk** — corrigiu de 4/10 para 9/10 queries acertadas no top-5 do BM25.
3. **Sinônimo WHP↔Whirlpool** no chunker (parcial — não cobriu o caso completo no smoke test).
4. **Tokenização tolera Unicode**: `≥`, `%`, `⭐` viram separadores; combining chars (acentos) são removidos preservando letra base.
5. **Síntese Claude é opcional** (`synthesize=True/False`) — testes rodam sem ANTHROPIC_API_KEY chamando só BM25.
6. **`@trace` Langfuse** ao redor da chamada Anthropic em `consulta_natural._call_claude_traced` (no-op se Langfuse desabilitado).
7. **Pipeline reporta "achados de qualidade"** — 1 template sem header inicial detectado tinha tipografia "Templete" + cabeçalho "atividade verbo objeto" — todos resolvidos com substring matching.

```python
def _normalize(s: str) -> str:
    import unicodedata
    s = unicodedata.normalize("NFKD", s)
    out = []
    for c in s:
        if unicodedata.combining(c):
            continue
        if ord(c) < 128:
            out.append(c)
        else:
            out.append(" ")
    return "".join(out).lower()
```
