## Bug conhecido
- `~/postcss.config.js` global referencia `tailwindcss` que não está instalado
  no monorepo zzyon-platform. Sintoma: Vite quebra em `Cannot find module 'tailwindcss'`.
  Workaround: vite.config.js de cada torre define `css.postcss.plugins=[]`.
