**Why:** Mac mini fica ligado 24/7 com Tailscale, ideal para serviços sempre-on. MacBook viaja com o usuário. Tailscale dá IPs internos estáveis sem expor a internet.

**How to apply:**
- Em qualquer .env de desenvolvimento, apontar para Mac mini Tailscale, NUNCA `localhost`.
  - `EVOLUTION_API_URL=http://100.65.23.99:8080`
  - `SUPABASE_URL=http://100.65.23.99:8100`
  - `LANGFUSE_HOST=http://100.65.23.99:3000`
- Para falar com Docker do Mac mini via SSH, usar `export PATH="/opt/homebrew/bin:$PATH"; export DOCKER_HOST="unix:///Users/rubens/.colima/default/docker.sock"` (Colima) ou `unix:///Users/rubens/.orbstack/run/docker.sock` (OrbStack).
- Para confirmar que Mac mini está alcançável: `tailscale status | grep mac-mini`.
